🔑 How to create a Grex-ID API token

Follow these steps inside your Grex-ID panel.

  1. Sign in to your Grex-ID account at the base URL above.
  2. Open the sidebar and go to Tokens.
  3. Click Create Token and give it a descriptive name (e.g. Camera Bridge).
  4. In the permissions field, add both of the following:
    grexid.detect grexid.list_locations
  5. Click Save. The raw token is shown once — copy it immediately.
  6. Paste it into the API Token field on this screen.

Without both permissions this bridge cannot list locations or submit detections.
